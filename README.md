# linconvidal-site

Static personal site for `linconvidal.com`.

Direction: a personal home for software, visual work, games, and notes/essays. Built as a static Astro site to avoid recurring hosting costs.

## First pillars

- `/visuals` — photo essays, zines, and selected print work.
- `/software` — Cobogó, Farol, SpatialMD, tools, research work.
- `/games` — Pinote and Cotinga Games.
- `/notes` — essays, field notes, devlogs, and blog-like writing.

## Commands

```bash
npm install
npm run dev
npm run build
```
